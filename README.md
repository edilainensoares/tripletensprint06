# tripletensprint06
Projeto de Banco de Dados

Nesse projeto teve como objetivo, os alunos trabalharem no console com logs e em um banco de dados para um aplicativo de táxi. 

As etapas foram: 
* Descobrir quais solicitações vieram de um endereço IP via console
* Movimentar os logs no console, conforme solicitações.
* Consultar quantidades de dados no Banco de Dados.
* Criar tabelas no Banco de Dados via comandos.
* Obter dados das tabelas do Banco de Dados.

Pojeto revisado e aprovado pelos instrutores.
